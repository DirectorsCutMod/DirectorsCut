# Director's Cut
A community-driven filmmaking tool built on Source.

Current development stage: **Pre-Alpha**

The goal is to recreate **Source Filmmaker** as an open-source tool for the **Source** community and eventually release as a standalone app on **Steam**.

## For Developers
Director's Cut is only intended for developers at this time.

Read building and debugging instructions in [BUILDING.md](BUILDING.md), this is the only way to obtain binaries at the moment.

See [USAGE.md](USAGE.md) for instructions on running Director's Cut through **Steam** or as an engine tool for your own mod.

Learn how to contribute at [CONTRIBUTING.md](CONTRIBUTING.md), and be sure to check out the [community](#links)!
